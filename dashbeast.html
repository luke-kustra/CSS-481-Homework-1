Project Overview

My role at Dashbeast Inc. was to create a Machine Learning Model that accepts

acceleration, gyroscopic, and magnetometer data in the X, Y, and Z directions.

Using that data the model will predict what exercise the user is performing,

such as a bench press, a squat, or a pushup. This model utilized a One-Dimensional

Convolutional Neural Network for temporal pattern recognition and classification.

Model Architecture

Technical Implementation

This model utilized a One-Dimensional Convolutional Neural Network Model.

Training Process Details

The training of which involves the stacking of four Conv1D blocks with BatchNorm, ReLU, and max-pooling to learn temporal patterns and downsample the sequence.

Then flattening to two fully connected layers with dropout before a final linear layer that outputs logits for each exercise class (trained with cross-entropy loss).

Model Output & Integration

Output Specification

The model outputs a .json file with the predicted workout alongside a confidence score.

"The .json file makes the ML feature module for other systems to interact with the model easily, enabling seamless integration across multiple platforms and applications."
